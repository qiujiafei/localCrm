# webpackflow

> 基于 webpack2 + react + stylus + HMR 的开发模版

## 特性

- [x] webpack2
- [x] react
- [x] material-ui
- [x] stylus
- [x] hot module replacement

## 在线 Demo

[点击这里](https://cntanglijun.github.io/webpackflow/)

## 克隆仓库

```bash
git clone https://github.com/cntanglijun/webpackflow.git
```

## 安装依赖模块 

```bash
npm i
```

## 启动

```bash
npm start
```

## 测试

```bash
npm test
```

## 打包

```bash
npm run build
```
